# Netology_test_case_traversing_the_matrix
Реализована Python-библиотека, осуществляющая получение квадратной матрицы (NxN) с удалённого сервера и возвращает её пользователю в виде List[int]. Этот список должен содержать результат обхода полученной матрицы по спирали: против часовой стрелки, начиная с левого верхнего угла.
